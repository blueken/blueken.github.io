serverless-stack-cn

serverless-stack.cn

发现国外的 serverless-stack 如火如荼，国内暂时没有找到一家专注 serverless 的网站或者论坛，干脆注册了 serverless-stack.cn

为什么不是 serverless-stack.com.cn

目标不仅仅是把 serverless-stack 上的好内容翻译一下。以后的以后还要为基于 Aliyun, Azure 等其它云服务商做对应的学习和教程

短期目标

暂时不会花精力去专门创建 Serverless-stack.cn 的网站， 会继续在本博客写关于 serverless 的内容 （包括serverless-stack上关于 AWS 的相关内容）

那么新的域名怎么办？ CNAME 到 jiafeifei.xyz/serverless-stack来

步骤一

mkdir /home/bob/serverless-stack && cd $_

vim index.html

步骤二

vim /etc/nginx/nginx.conf

nginx -s reload

步骤三

解决 vim 乱码问题

vim /etc/vimrc

步骤四

解决 页面 乱码问题

vim /home/bob/serverless-stack/index.html

收工！
