hello Mongoose

什么是 Mongoose

Mongoose 是为 Node.js 提供的优雅 MongoDB 对象模型。

说人话

其实就是 MongoDB NodeJS 版本的 ORM

Getting Started

  1. mkdir h-mongoose && cd $_
  2. npm init -y && npm install mongoose –save

  3. vi app.js

  4. node app.js

    connected :)

    saved! Huahu

  1. https://cloud.mongodb.com/v2 -> Clustrrs Overview -> COLLECTIONS -> 可以看到 test.kitties 有三条插入的记录
