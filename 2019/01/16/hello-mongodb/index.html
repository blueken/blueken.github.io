你好Mongodb

MongoDB 概念

SQL术语/概念 MongoDB术语/概念 解释/说明
database database 数据库
table collection 数据库表/集合
row document 数据记录行/文档
column field 数据字段/域
index index 索引
table joins 表连接,MongoDB不支持
primary key primary key 主键,MongoDB自动将_id字段设置为主键

MongoDB Atlas

MongoDB 的云服务，免费哦

  1. 打开官网， 点击 Start free
    https://www.mongodb.com/cloud/atlas?jmp=nav

  2. 右上角点击用户名称下拉菜单 -> Organizations -> +New Organization -> (Name=rest-api) Next -> Create Organization -> New Project -> (Name=rest-api) Next -> Create Project -> Build A Cluster -> Cloud Provider & Region 选择
    AWS, Singapore (ap-southeast-1) -> Create Cluster -> 等待5分钟 -> 创建成功后，从Overview 切换到 Security Tab -> ADD NEW USER -> 切回Overview Tab -> CONNECT -> ADD Your current IP Address -> Choose a connection Method -> Connect Your Applictioan -> Standard connection string -> Copy String -> close

Node.js 连接 MongoDB

  1. mkdir h-mongodb && cd $_
  2. npm init -y
  3. npm install mongodb –save

  4. vi app.js

  5. ➜ h-mongodb node app.js

    (node:60512) DeprecationWarning: current URL string parser is deprecated, and will be removed in a future version. To use the new parser, pass option { useNewUrlParser: true } to MongoClient.connect.
    Connected successfully to server
    Inserted 3 documents into the collection

  6. https://cloud.mongodb.com/v2 -> Clustrrs Overview -> COLLECTIONS -> 可以看到 firstdb.documents 有三条插入的记录
