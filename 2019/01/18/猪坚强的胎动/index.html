猪坚强的胎动

猪坚强是谁？

老婆怀孕28周了，按照医生的说法，从本周开始要正式数胎动了，以便观察胎儿的健康情况。

所以猪坚强是还未出生的小盆友，这是她妈妈给他起的非正式名字。

胎动记录

老婆每天会记录3次胎动，微信发给我，我记在 Magic2 的 WPS Office 里。最近对 Serverless 比较感兴趣，索兴用 Serverless 来实现一个简单的 WebApp

技术架构

  • UI: AntDesign + Recharts

  • 前端： GatsBy + Serverless + Finch + S3

    如果不在乎上传第三方平台的话，大力推荐 Surge = ServerlessFramework + Finch Plugin + S3。有工具，送域名，还送 https ，童叟无欺！

  • 后端: NodeJs + Serverless + AWS Lambda

  • 数据库：MongoDB + Mongoose + MongoDB Atlas

    开工大吉
